IGllama Documentation

Welcome to the neural interface. You’ve entered the control layer for IGllama, your gateway to local AI inference with GGUF models.

Georgi Gerganov Unified Format (GGUF) is the backbone here, a quantized model format that runs efficiently on consumer hardware. This isn’t cloud AI, this is yours.
